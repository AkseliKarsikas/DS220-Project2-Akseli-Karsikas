Olympic Athlete Data Analysis (18962016)

DS 220 Project #2: Data Analysis with Python Akseli Karsikas

This project explores long-term trends in Olympic athlete demographics and performance using a publicly available dataset hosted on Zenodo, covering all modern Olympic Games from 1896 to 2016.

Motivation

The Olympic Games provide one of the most complete long-term datasets in sports history. With athlete records spanning more than 120 years, this dataset makes it possible to study how athlete characteristics, participation, and medal outcomes have evolved over time. As someone who competes in elite gymnastics with the dream of making the Olympics one day, I was especially interested in exploring how factors such as age, height, weight, and gender vary by sport and decade.

Understanding these trends helps reveal how different sports have changed, which countries dominate specific events, and how athlete demographics reflect larger social and historical shifts in global sport. This makes the Olympic dataset a rich and meaningful subject for data analysis using Python.

Dataset Provenance

The dataset used in this project is the Full Olympics Dataset (athlete_events.csv) hosted on Zenodo, a public open-access repository operated by CERN and OpenAIRE. Zenodo assigns each dataset a DOI to ensure long-term accessibility and reproducibility.

The dataset includes athlete-level records for every modern Olympic Games from 1896 to 2016, including characteristics such as age, sex, height, weight, sport, event, team, NOC (country code), and medal results.

I downloaded the dataset on December 10, 2025 from the following link:

Zenodo Record: https://zenodo.org/records/11449966

Direct CSV Download: https://zenodo.org/records/11449966/files/athlete_events.csv?download=1

Final Insights

This project revealed several long-term trends in Olympic athlete demographics and performance. Athlete age, body size, and medal distribution show meaningful patterns across more than a century of Olympic history. Height and weight vary dramatically by sport, and Gymnastics in particular has distinctly smaller athletes. Countries with established sports infrastructure (USA, Russia, Germany) dominate the medal tables. Though developed smaller countries like Finland can rack up pretty impressing numbers of medals despite their small population.
